# AstraFlux — RTD + Repo Auditor

**What this repo gives you:**
1) **RTD (Real-Time Debugger):** send error logs → see live **Root Cause + Fix** on a dashboard.  
2) **Repo Auditor:** scan any local repo → auto-create **README.md, requirements.txt, .gitignore, Dockerfile, GitHub Actions CI** to make it **GitHub-ready & deploy-ready**.

---

## Quick Start (RTD)

```bash
npm install
npm start
# open http://localhost:8080
